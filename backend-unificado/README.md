# Backend Unificado

Backend unificado com compatibilidade para contratos legados.

## Executar

```bash
npm install
cp .env.example .env
npm run dev
```

## Objetivo

- Manter rotas legadas ativas durante migracao.
- Centralizar regras de dominio (receitas, pacientes, medicamentos, horarios).
- Padronizar seguranca, validacao e tratamento de erros.
